# Harrison Wells Brand Identity

Visual identity reference for LinkedIn content and graphics.

## Businesses

### DodoDigital.ai
**Tagline**: "We Build AI Systems That Drive Revenue"

**Target**: Austin small businesses, agencies, funds

**Color Palette**:
| Color | Hex | Usage |
|-------|-----|-------|
| Primary Blue | `#3B82F6` | CTAs, buttons, accents |
| Dark Navy | `#0F172A` | Headlines, primary text |
| Light Background | `#F9FAFB` | Page background |
| Pale Indigo | `#E0E7FF` | Gradient accents |
| White | `#FFFFFF` | Cards, clean space |

**Typography**:
- Font: Inter (system sans-serif)
- Headlines: Bold, dark navy
- Body: Regular weight, readable

**Visual Style**:
- Clean, minimalist, tech-forward
- Soft gradients (blue-50 to blue-100)
- Generous whitespace
- Professional but approachable
- Numbered step processes (1, 2, 3)
- Metric callouts ("15 hours saved", "$47K recovered")

**Key Messaging Themes**:
- Results-oriented, practical (not theoretical)
- Revenue-focused outcomes
- "No fluff, no theories"
- Austin local emphasis
- Quick diagnosis → proven solutions → fast implementation

## LinkedIn Content Style Guidelines

### For LinkedIn Carousels

**Recommended Color Palette** (combine both brands):
| Color | Hex | Usage |
|-------|-----|-------|
| Background | `#F9F9F7` | Off-white paper texture |
| Headlines | `#0F172A` | Dark navy/black |
| Accent 1 | `#3B82F6` | Blue highlights (DodoDigital style) |
| Accent 2 | `#2DD4BF` | Teal highlights (Joe M-C style) |
| Highlight | `#FEF08A` | Yellow marker on metrics |
| Emphasis | `#EF4444` | Red underlines |

**Typography**:
- Headlines: Bold serif (for carousel style) or bold sans-serif (for brand consistency)
- Body: Clean sans-serif
- CTAs: Italic, lighter weight

**Visual Elements**:
- Hand-drawn arrows (anti-AI aesthetic)
- Paper texture backgrounds
- Numbered steps
- Metric callouts with highlights
- Personal headshot + handle branding

**Content Themes That Fit Brand**:
- AI implementation tips
- Revenue/ROI-focused insights
- Practical workflows and automations
- "Here's what actually works" angle
- Austin business insights
- Executive AI adoption
