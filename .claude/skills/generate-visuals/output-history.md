## Style Checklist

Before creating new content, review recent outputs above. Aim for variety in:

- [ ] **Color accent**: Blue, teal, green, orange, purple
- [ ] **Background**: Paper texture, solid color, gradient, photo-based
- [ ] **Typography**: Serif headlines, sans-serif headlines, mixed
- [ ] **Layout**: Centered, left-aligned, asymmetric
- [ ] **Visual elements**: Hand-drawn arrows, geometric shapes, icons, photos
- [ ] **Topic diversity**: Don't repeat same subject matter back-to-back
